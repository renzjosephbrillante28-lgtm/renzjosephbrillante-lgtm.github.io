<h2 class="mid">Structure of CSS</h2>   
<img class="mtr" src="struc.png" alt="structure of css">
<p><b>Selector</b> – the HTML element you want to style; is a
 pattern used to "find" or "select" the HTML
 elements on a webpage to which a set of
 CSS rules will be applied.<br>
<b>Property</b> – the style attribute you want to modify; is a
 named characteristic or attribute that
 defines a specific aspect of how an HTML
 element should be displayed or rendered by
 the browser. Properties are used within
 CSS declarations to apply styles to selected
 elements<br>

<b>Value </b>– the weight of the attribute you want to apply to the
 element property; is the specific setting or
 configuration assigned to a CSS property. A property
 and its value form a CSS declaration, which dictates
 how an element should be styled<br>
<b>Declaration</b> – is a single instruction that defines a specific
 style for an element<br>
<b>Declaration block</b> – fundamental part of a CSS rule. It is a
block of code that contains one or more CSS declarations,
which define the styling rules for the selected HTML
element(s)</p>

<h2>The Style Sheet and its Parts</h2>
<p>• A set of instructions to a Web browser on how to
 display various elements on a Web page is known as
 <strong>stylesheet.</strong><br>
• Every CSS (whether it is contained in a .css file, or
 embedded in the head element of an HTML
 document) is a series of instructions called
 <strong>statements</strong></p>

• easily allow you to change the layout of your entire
web site by simply changing the external style sheet
and every page is instantly updated<br>
• has a .css file extension </p>

<h3>Demand Function</h3>

      <p>-Matematikal formula ng Demand</p>
      
      
<p><b>Equation: (Qd=a-bP)</b><br> 
Mga bahagi Bahagi Kahulugan<br>
-<b>Qd Quantity Demanded</b> (Dami ng Demand). Ito ang dependent variable, na nangangahulugang ang halaga nito ay nakasalalay sa mga pagbabago sa presyo.<br>
-<b>a Autonomous Demand</b> (Dami ng Demand sa Presyong Zero). Ito ang quantity demanded kapag ang presyo (P) ay 0. Sa konteksto ng ekonomiya, kinakatawan nito ang lahat ng di-presyong salik na nakakaapekto sa demand (tulad ng kita, panlasa, at populasyon).<br>
-<b>b Slope ng Demand Curve</b>. Ito ay isang negatibong numerong nagpapakita kung gaano kababa ang dami ng demand tuwing tumataas ang presyo. Ito ang sumasalamin sa Law of Demand, na nagsasaad na may baliktad na ugnayan sa pagitan ng presyo at dami ng demand.<br>
-<b>P Price</b> (Presyo). Ito ang independent variable sa function.</p>

<h3>Supply Function</h3>

      <p>-Matematikal formula ng Demand</p>
      
      
<p><b>Equation: (Qd=c+dP)</b><br> 
Mga bahagi Bahagi Kahulugan<br>
-<b>Qs Quantity Supplied</b> (Dami ng Supply). Ito ang dependent variable, na nagbabago batay sa presyo.dSlope ng Supply Curve. Ito ang positibong numerong nagpapakita kung gaano karami ang pagbabago sa dami ng supply sa bawat pagbabago sa presyo. Ito ay sumasalamin sa positibong ugnayan sa pagitan ng presyo at supply.<br>
-<b>c Y-intercept</b> (Supply sa Presyong Zero). Ang theoretical quantity supplied kapag ang presyo P ay 0. Sa praktikal na kahulugan, ito ay kumakatawan sa mga di-presyong salik na nakakaimpluwensya sa supply, tulad ng teknolohiya at gastos sa produksyon.<br>
-<b>d Slope ng Supply Curve</b> Ito ang positibong numerong nagpapakita kung gaano karami ang pagbabago sa dami ng supply sa bawat pagbabago sa presyo. Ito ay sumasalamin sa positibong ugnayan sa pagitan ng presyo at supply.<br>
